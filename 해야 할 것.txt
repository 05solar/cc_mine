
파일 업로드 팝업 만들기 - 내 컴퓨터에서 선택 창 띄우기 - 확인 누르면 이벤트 (여러 개 선택 가능하게할지?)
파일/폴더 메뉴 클릭 시 드롭다운 메뉴 만들기{
  파일 다운로드 / 폴더는 다운로드 안됨 or 폴더를 압축해서 다운로드
  파일 이동 / 폴더 이동(내부까지 전부 이동)
  파일 복사 / 폴더 복사(내부까지 전부 복사)
  파일 삭제 / 폴더 삭제(내부까지 전부 삭제)
  파일 속성(이름, 크기, 유형, 마지막 수정 날짜) / 폴더 속성(간단하게 이름, 크기만 하면 되지 않을까)
}

검색창 드롭다운 만들기 
설정 만들기 (일단은 다크모드, 알림설정 넣어놨는데 바꾸면 됨)
휴지통 만들기 (휴지통을 아예 따로 스토리지에 만드는 걸로)
클라우드 박스 로고 클릭 시 (start창으로 가기 vs 현재 창 새로고침) 효과 만들기

새로고침 기능 만들기 (페이지 새로고침 시 저장소 데이터를 최신화 (유저 및 설정은 그대로 저장소만 확인))
